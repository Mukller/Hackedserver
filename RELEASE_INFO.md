## Установка

### Пошаговая инструкция

1. **Скачайте JAR файл**
   ```bash
   wget https://github.com/Mukller/Hackedserver-/releases/download/v3.16.1/hackedserver-3.16.1.jar
   ```

2. **Поместите в папку plugins**
   ```bash
   cp hackedserver-3.16.1.jar /path/to/server/plugins/
   ```

3. **Перезагрузите сервер**
   ```bash
   # В консоли сервера:
   /reload
   
   # Или перезагрузите сам сервер:
   ./stop.sh
   sleep 5
   ./start.sh
   ```

4. **Проверьте установку**
   ```bash
   # В консоли сервера:
   /hs status
   
   # Вывод: "HackedServer 3.16.1 is running"
   ```

### Установка на Docker

```dockerfile
FROM openjdk:17-slim

WORKDIR /server

RUN apt-get update && apt-get install -y wget

# Скачиваем Paper server
RUN wget https://launcher.mojang.com/v1/objects/.../server.jar

# Копируем HackedServer плагин
COPY hackedserver-3.16.1.jar plugins/

# Запускаем сервер
CMD ["java", "-Xmx2G", "-jar", "server.jar", "nogui"]
```

## Обновление

### Обновление с предыдущей версии

#### Автоматическое обновление (рекомендуется)

```bash
# Используйте встроенную команду обновления
/hs update
```

#### Ручное обновление

1. **Остановите сервер**
   ```bash
   /stop
   ```

2. **Скачайте новую версию**
   ```bash
   wget https://github.com/Mukller/Hackedserver-/releases/download/v3.16.1/hackedserver-3.16.1.jar
   ```

3. **Замените старый файл**
   ```bash
   # Создайте резервную копию старой версии
   cp plugins/hackedserver-*.jar plugins/backups/
   
   # Замените на новую версию
   cp hackedserver-3.16.1.jar plugins/
   ```

4. **Запустите сервер**
   ```bash
   ./start.sh
   ```

5. **Проверьте логи**
   ```bash
   tail -f logs/latest.log | grep HackedServer
   ```

### Откат на предыдущую версию

```bash
# Восстановите из резервной копии
cp plugins/backups/hackedserver-3.15.8.jar plugins/hackedserver-3.15.8.jar

# Измените название файла
mv plugins/hackedserver-3.16.1.jar plugins/hackedserver-3.16.1.jar.old

# Перезагрузите сервер
/reload
```

## Совместимость между версиями

### Версия 3.16.x → 3.15.x (Откат)

⚠️ **Не рекомендуется** из-за различий в структуре конфигурации.

**Что может сломаться:**
- Новые параметры конфигурации будут потеряны
- Логи могут быть несовместимы
- Белые списки могут не работать

### Версия 3.15.x → 3.16.x (Обновление)

✅ **Полная совместимость** с автоматической миграцией конфигурации.

**Что будет перенесено:**
- Все параметры конфигурации
- Черные/белые списки игроков
- История логов
- Настройки уведомлений

## Решение проблем при установке

### Проблема: Плагин не загружается

**Решение:**
1. Проверьте, что файл находится в `plugins/`
2. Проверьте права доступа: `chmod 644 hackedserver-3.16.1.jar`
3. Проверьте версию Java: `java -version`
4. Посмотрите логи: `tail -100 logs/latest.log`

### Проблема: "Could not load plugin"

**Решение:**
```bash
# Проверьте, что это не модификация плагина
md5sum hackedserver-3.16.1.jar

# Переустановите плагин
rm plugins/hackedserver-3.16.1.jar
wget https://github.com/Mukller/Hackedserver-/releases/download/v3.16.1/hackedserver-3.16.1.jar
/reload
```

### Проблема: Низкая производительность

**Решение:**
1. Увеличьте выделенную память серверу:
   ```bash
   # Измените в start.sh на:
   java -Xmx4G -Xms4G -jar server.jar nogui
   ```

2. Отключите ненужные проверки в конфигурации:
   ```yaml
   # plugins/HackedServer/config.yml
   detection:
     aimbot: false      # если не нужно
     reach-hack: false  # если не нужно
   ```
